# AMRO Market Vision Core v2

Purpose:
This package defines the foundation layer for Brain-2 inside AMRO.

Brain-2 role:
- interpret market behavior
- translate visual structure into behavioral context
- retrieve historical similarity
- estimate probability
- compress context for governance layer

NOT:
- signal generator
- autonomous prediction engine
- direct execution engine

Architecture Position:
Brain-1 -> Observation Layer
Brain-2 -> Interpretation Layer
Brain-3 -> Governance Layer

Core Requirement:
All emotional interpretation must remain:
- measurable
- replayable
- auditable
- probability-based

No narrative-only interpretation allowed.

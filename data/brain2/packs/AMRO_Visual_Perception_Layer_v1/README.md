# AMRO Visual Perception Layer v1

Purpose:
Create the first visual perception layer for Brain-2.

Core Objective:
Allow Brain-2 to:
- recognize visual market behavior
- interpret emotional structure
- compare historical similarity
- detect emotional transition
- identify multi-timeframe conflict

Important:
This package is not a prediction system.

It is:
- visual perception memory
- behavioral interpretation support
- replay-oriented context layer

Main Components:
- chart snapshot memory
- annotated behavior mapping
- transition replay memory
- visual similarity memory
- multi-timeframe emotion mapping
